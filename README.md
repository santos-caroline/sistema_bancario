![Python](https://img.shields.io/badge/Python-3.9.13-blue?style=flat&logo=python)
![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-yellow)


O projeto foi desenvolvido seguindo as diretrizes do desafio de código do **Bootcamp Suzano - Python Developer** da **DIO**.  
A estrutura original foi ajustada para enfatizar o uso de **estruturas de dados** e **laços de repetição**, com o objetivo de aprofundar meus aprendizados na linguagem e tornar a lógica do sistema mais organizada e eficiente.


## Sobre o Projeto
O **Sistema de Gerenciamento de Contas Bancárias** é uma aplicação em linha de comando que simula operações essenciais, como **depósito**, **saque** e **emissão de extrato**.

O projeto prioriza:
- **Boas práticas de programação**
- **Modularização e organização do código**
- **Controle de fluxo** com condicionais e laços

## Funcionalidades:
-  **Sacar**: Operação conforme saldo disponível, com limite diário e validação de valores.
-  **Depositar**: Adiciona valores ao saldo com verificação de entrada.
-  **Extrato**: Lista todas as movimentações de forma clara e organizada.

  
## Tecnologias Utilizadas:
- **Python 3.9.13** — Linguagem de programação principal.
- **Módulo `datetime`** — Registro de data e hora das operações.
- **Entrada e saída padrão (`input` / `print`)** — Interface simplificada via linha de comando.
- **Estruturas de dados nativas (`list`, `dict`)** — Armazenamento de operações e controle de fluxo.
- 

## Pré-requisitos Recomendados:
- Uma IDE como VS Code ou PyCharm
- Um terminal/prompt de comando
- **Ambiente virtual (`venv`)**

## Como Executar

### 1. Clone o repositório:
```bash
git clone https://github.com/santos-caroline/sistema_bancario.git
```

### 2. Acesse o diretório do projeto:
```bash
cd sistema_bancario
```

### 3. Crie um ambiente virtual (Opcional)
```bash
python -m venv venv
```
- Ative o ambiente no Windows:
```bash
venv\Scripts\activate  
```
- Ative o ambiente no Linux/Mac:
```bash
source venv/bin/activate  
```
 
### 4. Execute a aplicação:
```bash
python desafio_dio_sistema_bancario.py 
```
### Ou, se estiver usando VS Code:

- Abra a pasta do projeto (File > Open Folder).

- Abra o arquivo principal .py

- Clique em ▶ Run Python File no topo da interface.
